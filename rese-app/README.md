# Rese 飲食店予約アプリ
<img width="881" alt="image" src="https://user-images.githubusercontent.com/118349010/232480305-68fafeff-1164-4aa4-8c35-dc40c8204c38.png">
## 作成した目的
模擬案件

## 機能一覧
- ログイン
- 会員登録
- 飲食店検索
- 飲食店詳細画面表示
- お気に入り店舗登録
- マイページ機能（予約管理・お気に入り一覧・来店店舗評価）

## 使用技術（実行環境）
- Laravel Framework 8.83.27 / Mysql
